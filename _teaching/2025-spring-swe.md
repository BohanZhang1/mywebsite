## CSCI 5030: Principles of Software Development
**Saint Louis University, Spring 2025**

---

### Guest Lecture: Version Control Essentials: Git and GitHub in Practice

An interactive lecture introducing graduate students to professional version control workflows using Git and GitHub. The session emphasized practical skills used daily in both industry and research settings.

**Topics covered:**

- Why version control is non-negotiable in collaborative software development
- Core Git concepts: commits, branches, staging, and the working tree
- GitHub workflows: repositories, pull requests, and code review
- Branching strategies for team projects
- Resolving common issues: merge conflicts, rebasing, and history management
- Hands-on demonstration using a live repository
