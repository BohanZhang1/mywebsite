## CSCI 5300: Software Engineering
**Saint Louis University, Fall 2026**

Two invited hands-on workshops delivered to a graduate-level software engineering course.

---

### Workshop 1: Docker and Containerization

A hands-on session taking students from zero Docker experience to running, configuring, and persisting a containerized web application. Covered images, containers, port mapping, environment variables, volume mounts, networking, Docker Compose, and pushing to Docker Hub.

---

### Workshop 2: Continuous Integration with GitHub Actions

A practical session where students built a complete CI pipeline from scratch using a Python project. Covered workflow file authorship, triggers, runners, Marketplace actions, and the red/green commit cycle.
