Presented the tool demonstration paper for DroneReqValidator (DRV) at ASE 2023. The demo showcased DRV's three-step wizard interface for configuring realistic simulation environments, specifying UAV sensor configurations, and setting safety test properties. Live demonstrations highlighted DRV's automated fuzzy testing capabilities, runtime monitoring, and acceptance test report generation for sUAS developers.

**Authors:** Bohan Zhang, Yashaswini Shivalingaiah, Ankit Agrawal

**Conference:** ASE 2023 — Echternach, Luxembourg, November 11–15, 2023. Pages 2082–2085.
